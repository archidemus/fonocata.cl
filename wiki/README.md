# Wiki Fonocata

Fuente de verdad para la marca, contenido y presencia digital de Catalina Orellana Molnar (FonoCata).

## Secciones

| Archivo | Qué contiene |
|---------|-------------|
| [marca.md](marca.md) | Identidad visual, voz, tono, paleta de colores |
| [bio.md](bio.md) | Biografías oficiales (corta, media, larga) |
| [servicios.md](servicios.md) | Descripción detallada de cada servicio |
| [formacion.md](formacion.md) | Títulos, diplomados, certificaciones |
| [redes/README.md](redes/README.md) | URLs, handles, estrategia por red social |
| [contenido.md](contenido.md) | Plan de contenido, hashtags, categorías de posts |

## Principios

1. **Trato humano** — todo el contenido refleja empatía y cercanía
2. **Evidencia clínica** — información basada en ciencia, sin jerga innecesaria
3. **Consistencia** — misma voz, misma paleta, mismo mensaje en todas las plataformas
4. **Español neutro** — comprensible en todo Chile y LatAm
